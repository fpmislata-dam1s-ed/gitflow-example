# Gitflow example
Aquest document inclou el temari que pot eixir a l'examen de la 3a avaluació
del mòdul __Entorns de Desenvolupament__ de __DAMs1__.

Les unitats didàctiques estudiades són les següents:
- __UD6:__ Documentació
- __UD7:__ Diagrames UML
- __UD8:__ GitFlow

## UD6: Documentació
- Documentar codi Java amb Javadoc
- Generar la documentació
- Publicar documentació a GitHub Pages

## UD7: Diagrames UML
- Diagrama de casos d'ús
- Diagrama de seqüència
- Diagrama de comunicació
- Diagrama d'activitats
- Diagrama d'estats

## UD8: Gitflow
- Conèixer i saber aplicar l'estratègia de ramificació Gitflow
- Incorporar els canvis mantenint una història linial.
- Incorporar les funcionalitats mitjançant `squash`
- Forks i Pull Requests
